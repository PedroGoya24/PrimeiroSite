Sobre o Aplicativo

PAEST (Projeto de Apoio ao Estudante)

Telas do Projeto

Aqui, apresentaremos a você, cada tela presente no Aplicativo e suas funções. E logo após, disponibilizaremos um link para o download do PAEST (Projeto de Apoio ao Estudante).

Login


Neste é a concretização do cadastro; após efetuar o cadastro o usuário deve digitar 2 campos - email e senha - para conceber o acesso ao aplicativo.


Cadastro


Aqui é realizado o seu registro em nosso aplicativo, nele o usuário deve cadastrar-se utilizando seu nome completo, email, celular e senha.


Menu



Está área é reservada para a visualização integral do projeto, isto é, para uma melhor lucidez quanto as possibilidades dos elementos constituintes do aplicativo que podem ser realizadas pelo usuário. Aqui será encontrado os próximos campos: rotina, diário, alimentação, site e informativo.


Rotina


Este campo será destinado para atividades como um todo, entre elas estão: Atividades físicas, meditações, fé, hobbie e alimentação. Todas as elas tem como fim uma respectiva função de suma relevância para a construção de uma mente saudável.


Alimentação



Este campo será destinado para atividades como um todo, entre elas estão: Atividades físicas, meditações, fé, hobbie e alimentação. Todas as elas tem como fim uma respectiva função de suma relevância para a construção de uma mente saudável.


Informativo



Neste campo é dedicado a informação do usuário, para que o mesmo crie consciência da visceral e tenebrosa desestabilização mental e adquira conhecimento sobre.


Perfil



No perfil é atribuído as suas informações pessoais, a opção de sair da sua conta e por fim o contato da Resilience Corporation, caso haja dúvidas, aforismos, problemas com o aplicativo entre outros.
